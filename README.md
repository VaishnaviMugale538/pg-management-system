# PG Management System

Full Stack PG Management Platform built using React and Spring Boot.

## Tech Stack

* React.js
* Spring Boot
* MySQL
* REST APIs
* Chart.js

## Features

* Admin Dashboard
* Tenant Dashboard
* Room Management
* Rent Payment Tracking
* Payment History
* Pending Rent Alerts
* Revenue Analytics
* Expense Tracking

## Future Improvements

* Online Payment Gateway
* Email Notifications
* Mobile App
